🤖 给 AI 的指令（让团队的 AI 自动按方法论干活）

仓库里有两个 AI 指令文件，对方 clone 后 AI 自动读、无需配置：
· Codex / 通用约定 读 AGENTS.md（核心指令：角色、工作循环、写文档规则、铁律）
· Claude Code 读 CLAUDE.md（速记 + 转发到 AGENTS.md）

怎么开始：把整套规范 当新项目起点，对方第一句直接说 —— “按 AGENTS.md 的方法论，帮我写 overview 总纲”，AI 就会照模板和规则开干。
换 Cursor / Copilot / Windsurf？加一行转发文件指向 AGENTS.md 即可（见 README.md）。

⚡ 5 分钟上手

  1. Copy 整个目录到新项目（或单独建 doc 仓库）。
  2. 全局搜 --brand，把配色 token 换成你的品牌色。
  3. 写 docs/overview.html 业务总纲（回答 5 个问题）—— 写不清＝还没想清。
  4. Copy 场景 / 设计模板，每篇画 Mermaid + 写铁律 + 列可配参数。
  5. 更新 docs/index.html 导航卡 + docs/feature-status.html 状态表。
  6. 双击 docs/index.html 看效果（file:// 直接可看，无需起服务）。
顺序很重要：总纲先行。 别上来建一堆空壳。
